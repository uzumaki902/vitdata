build this in React Native + Expo and apply 60/30/10 UI rule.

1) App flow (simple)

Login Screen: email + password + login button

On success → Dashboard Screen

Dashboard shows: Name, Email, Age

For now you can use dummy login (hardcoded user) unless backend is given.

2) 60/30/10 UI rule (how to apply here)

60% = Base / Background

Use a soft neutral: #F6F7FB (light grey) or #FFFFFF

Most of the screen stays this color.

30% = Secondary (cards, surfaces, sections)

Use white cards on grey background OR light tint blocks.

Example card bg: #FFFFFF with shadow + rounded corners.

10% = Accent (buttons, highlights, focus states)

Use 1 strong color only for actions: e.g. #4F46E5 (indigo)

Login button, active input border, small icons.

Rule: Don’t use accent for everything—only “important action”.

3) Recommended layout + spacing

Use one main card in login (clean)

Use 2–3 cards in dashboard (profile + info)

Spacing: 16 or 20 consistently

Border radius: 12–16

Font sizes: Title 22–24, labels 12–14, inputs 16

4) Minimal Expo setup (navigation)

Use expo-router (best with Expo) OR React Navigation.
I’ll give you expo-router way (cleaner).

Folder structure

app/index.js (Login)
import { useState } from "react";
import { View, Text, TextInput, Pressable, StyleSheet } from "react-native";
import { useRouter } from "expo-router";

export default function Login() {
  const router = useRouter();
  const [email, setEmail] = useState("");
  const [pass, setPass] = useState("");
  const [error, setError] = useState("");

  const onLogin = () => {
    // Dummy auth (replace with real API later)
    if (email.trim() === "test@mail.com" && pass === "1234") {
      setError("");
      router.push({
        pathname: "/dashboard",
        params: { name: "Sai", email: email.trim(), age: "21" },
      });
    } else {
      setError("Invalid email or password");
    }
  };

  return (
    <View style={styles.page}>
      <Text style={styles.title}>Welcome back</Text>
      <Text style={styles.sub}>Login to continue</Text>

      <View style={styles.card}>
        <Text style={styles.label}>Email</Text>
        <TextInput
          value={email}
          onChangeText={setEmail}
          placeholder="you@example.com"
          keyboardType="email-address"
          autoCapitalize="none"
          style={styles.input}
        />

        <Text style={[styles.label, { marginTop: 12 }]}>Password</Text>
        <TextInput
          value={pass}
          onChangeText={setPass}
          placeholder="••••"
          secureTextEntry
          style={styles.input}
        />

        {!!error && <Text style={styles.error}>{error}</Text>}

        <Pressable style={styles.btn} onPress={onLogin}>
          <Text style={styles.btnText}>Login</Text>
        </Pressable>
      </View>
    </View>
  );
}
